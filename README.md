# 看護略語 検索AIアシスタント

無料のローカル埋め込み検索（TF-IDFベクトル）で候補を抽出し、必要に応じてGeminiに回答生成させるStreamlitアプリです。

## 特徴
- 無料範囲で使える埋め込み検索: `scikit-learn` のTF-IDF + コサイン類似度
- RAG風の安全設計: 見つかった候補行のみをプロンプトに渡す
- APIキーなしでもテストモードで動作
- スタイリッシュなUI（グラデーション背景・カード・カスタムフォント）

## ファイル構成
1. `app.py`: アプリ本体
2. `data.csv`: 略語辞書データ
3. `requirements.txt`: 依存ライブラリ

## セットアップ

### 1. 依存ライブラリをインストール
```bash
pip install -r requirements.txt
```

### 2. アプリを起動
```bash
streamlit run app.py
```

## 使い方
1. サイドバーでデータを確認
2. 質問欄に略語や症状を入力
3. `検索してAIに聞く` を押す
4. 埋め込み検索で抽出された候補と、AI回答を確認

## Gemini APIについて
- サイドバーに `AIza...` 形式のキーを入れるとGemini呼び出しを実行
- 未入力時はテストモードで回答例を表示
