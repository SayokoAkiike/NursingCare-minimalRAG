# 看護略語 検索AIアシスタント 準備手順

このフォルダには、アプリの設計図（ソースコード）が入っています。
あとでPythonなどをインストールしたときに、すぐに動かせるようになっています！

## ここにあるファイル
1. `app.py` : アプリを動かすメインのプログラムコードです。
2. `data.csv`: スプレッドシートの代わりとなるサンプルの辞書データです。
3. `requirements.txt`: アプリを動かすために必要な追加ツールの一覧です。

## あとで環境を用意したあとの動かし方

### 1. 必要なツールを入れる
コマンドプロンプト（またはTerminal）を開いて、以下のコマンドを入力し、必要なツールをインストールします。

```bash
pip install -r requirements.txt
```

### 2. アプリを起動する
以下のコマンドを入力すると、自動的にブラウザが立ち上がり、アプリの画面が開きます！

```bash
streamlit run app.py
```

### 3. 本物のOpenAIに繋ぎたくなったら
1. OpenAIのAPIキー（`sk-...`）を取得します。
2. `app.py` をメモ帳やVS Codeで開きます。
3. 5〜6行目にある `# from openai import ...` の行の `#`（コメントアウト）を消して、取得したAPIキーを貼り付けます。
4. 73行目付近にある `"""` （3つのダブルクォーテーション）を消すと、本物のAIが動くようになります！
